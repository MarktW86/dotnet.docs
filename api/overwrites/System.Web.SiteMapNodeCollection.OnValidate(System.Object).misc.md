<p>The <xref href="System.Web.SiteMapNodeCollection.OnValidate(System.Object)"></xref> method allows implementers to define processes that must be performed when executing the standard behavior of the underlying <xref href="System.Collections.ArrayList"></xref> object. By defining the <xref href="System.Web.SiteMapNodeCollection.OnValidate(System.Object)"></xref> method, implementers can add functionality to inherited methods without having to override all other methods. Specifically, the <xref href="System.Web.SiteMapNodeCollection.OnValidate(System.Object)"></xref> method is called anytime the <xref href="System.Web.SiteMapNodeCollection"></xref> collection is used via a reference to an <xref href="System.Collections.IList"></xref> interface.</p>
