<p>When you inherit from the <xref href="System.Web.UI.Adapters.PageAdapter"></xref> class, you can override the <xref href="System.Web.UI.Adapters.PageAdapter.CacheVaryByParams"></xref> property to return a list of the additional GET or POST parameters to control target-specific caching. The <xref href="System.Web.UI.Adapters.PageAdapter.CacheVaryByParams"></xref> base method returns `null`.</p>
