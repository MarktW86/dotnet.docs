<p>The <xref href="System.Windows.ContentElement"></xref> class does not yet define all aspects of a complete content element for flow-style presentation. <xref href="System.Windows.FrameworkContentElement"></xref> is an immediately derived class of <xref href="System.Windows.ContentElement"></xref>. <xref href="System.Windows.FrameworkContentElement"></xref> includes a more complete set of additional members that support rendering a <xref href="System.Windows.FrameworkContentElement"></xref> within a content host and using the WPF framework-level layout system.</p>
